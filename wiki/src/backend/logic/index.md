# 🧠 Backend Core Logic

## 📁 상세 항목

- [🎯 **Recommendation Logic**](./recommendation_logic.md): 전체 추천 서비스의 실행 흐름 및 단계별 로직.
- [🌈 **Visual-Scent Mapping**](./visual_scent_mapping.md): 시각 키워드를 향기 아우라 수치로 변환하는 매핑 규칙.

- [🧯 **Error Handling & Fallback**](./error_handling_fallback.md): 외부 API 장애 시 서비스 연속성을 보장하는 예외 처리 및 우회 전략.
